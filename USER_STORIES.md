# Brick Oracle — User Stories

## User

### Search Bar
> As a user, I want the ability to search on LEGO bricks, sets, and collections with filter options so that I can find a particular brick, set or collection that exists within the application's database.
- By theme filter
- Filters (general)

### Upload Collection
> As a user, I want to upload a collection of individual bricks or an entire set so that I can add them to my digital LEGO collection.

### Collection Collab
> As a user, I want to be able to combine collections together so that I can see what LEGO sets me and my friends can build together.


### Collection Share
> As a user, I want to be able to make my collection private or public so that I can share what information I want with the community.

### User Follow / Notifications
> As a user, I want to be able to follow users, collections, sets, etc. so that I can get notifications when there are updates to that record type if changes are made.

### User Comments
> As a user, I want to be able to comment on a user, collection, sets, etc. so that I can collaborate, help, and interact with the community.

### Brick Usage
> As a user, I want to be able to flag what bricks I am currently using in a set to indicate that I cannot actively use those bricks in another set so that I do not get recommended sets that I cannot build without breaking apart another set.

### Collection Storage
> As a user, I want to sort, categorize, and store my collection bricks into bins so that I can organize my collection and find pieces more easily.

### Create / Upload Custom Sets
> As a user, I want to upload custom sets so that I can track which bricks those use.

### User Login
> As a user, I want to be able to login to my profile which contains all my sets and collections so that my data is separate and stored.

### Non-Login User Path
> As a user, I want to be able to view the site without login so that I can know if this is worth my time or is interesting.

### Brick Diff
> As a user, I want to see what bricks I am missing from my collection against a set so that I can acquire those missing bricks to complete the set.

### Collection Viewer
> As a user, I want to be able to view my own collections and others so that I can engage with the community and see my own bricks.
- Popular or trending collections

### Set Viewer
> As a user, I want to be able to view sets, the bricks associated, as well as get recommended the popular ones so that I can understand what bricks are in a given set.

- Popular or trending sets

### Set Recommendations
> As a user, I want to be able to view set recommendations that are generated based on my collection so that I can find builds that are close to being completed or would be cheapest to complete.
- See which sets would be cheapest to complete

### Import from External Sources
> As a user, I want to be able to import sets or collections from other sites that provide a similar service so that I don't have to manually enter in all the bricks.
- Supported sources: Rebrickable, BrickLink, CSV

### Set Wishlist
> As a user, I want to wishlist specific sets so that I can compare my collection against the wishlist.

### Price Estimates
> As a user, I want to see market prices for individual bricks or full sets so that I can get quick cost estimates for completing a set or trading.
- Uses BrickLink / BrickOwl market prices

### Set Progress Tracker
> As a user, I want to see the progress I have to complete a set with bricks alone so that I can see which sets I am most close to completing.

### Builds Timeline
> As a user, I want to see the history of my builds and when I add bricks and complete sets so that I can see all the hard work I put into my collection.

### General Accessibility
> As a user, I want the site to be accessible to my disability so that I can interact with the community and store my LEGO collection.
- Colorblind, blind, and other accessibility considerations

### Camera Scan
> As a user, I want to be able to take a picture of my LEGO collections to upload them so that I do not have to manually upload bricks or sets.

### CAD Viewer
> As a user, I want to digitally see my sets, collection, and build sets so that I can view them in a different way.

---

## Admin

### Flag / Correct Inaccurate Sets
> As an admin, I want to flag or correct inaccurate sets so that users do not get access to those sets and have correct sets represented in search and viewer.

### Moderation
> As an admin, I do not want users to add content that is inappropriate so that the site is welcoming to everyone.

### Collection Database Management
> As an admin, I want to be able to modify the brick DB so that the catalog is up-to-date with LEGO.
